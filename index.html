📘 Análise Matemática Detalhada (Projeto 05)

   

1. Matriz de Adjacência ($M_{metrô}$)

   

Representação direta das conexões. Ordem: Avenida (A), Centro (C), Praça (Pç), Parque (Pq), Shopping (S), Terminal (T).

    
         A  C Pç Pq S  T
       ------------------
    A |  1  0  1  0  0  0 |
    C |  1  1  1  0  0  0 |
    Pç|  0  0  1  1  0  0 |
    Pq|  0  0  1  1  1  0 |
    S |  1  1  0  0  1  0 |
    T |  0  0  0  0  0  1 |
   
   

2. Classificação da Relação

   
           
  • Reflexividade: SIM ✅ (Diagonal principal é toda '1'.)
    •        
  • Simetria: NÃO ❌ (Exemplo: C → A é 1, mas A → C é 0.)
    •        
  • Assimetria: NÃO ❌ (A diagonal contém '1's.)
    • section      
  • Antissimetria: NÃO ❌ (Há caminhos de "mão dupla" entre Pç ↔ Pq.)
    •        
  • Transitividade: NÃO ❌ (Exemplo: S → C e C → Pç são 1, mas S → Pç é 0. Faltam atalhos.)
    •    
   

3. Fechos das Relações (Otimizações)

   

Cálculo da rede mínima necessária para possuir a propriedade desejada.

   

a) Fecho Simétrico ($S_{metrô}$)

   

A rede ideal se todas as rotas fossem bidirecionais.

    
         A  C Pç Pq S  T
       ------------------
    A |  1  1  1  0  1  0 |
    C |  1  1  1  0  1  0 |
    Pç|  1  1  1  1  0  0 |
    Pq|  0  0  1  1  1  0 |
    S |  1  1  0  1  1  0 |
    T |  0  0  0  0  0  1 |
   
   

b) Fecho Transitivo ($T_{metrô}$)

   

A rede com todos os atalhos diretos possíveis (mostra o que é alcançável).

NT     
         A  C Pç Pq S  T
       ------------------
    A |  1  1  1  1  1  0 |
    C |  1  1  1  1  1  0 |
    Pç|  0  0  1  1  1  0 |
    Pq|  0  0  1  1  1  0 |
    S |  1  1  1  1  1  0 |
    T |  0  0  0  0  0  1 |

4. Composição Metrô-Ônibus ($M_{composta}$)

   

Resultado da operação $M_{metrô} \circ M_{ônibus}$, para analisar a integração com uma baldeação.

   

Matriz de Ônibus ($M_{ônibus}$)

    
         A  C Pç Pq S  T
       ------------------
    A |  0  1  0  0  0  1 |
    C |  1  0  0  1  0  0 |
    Pç|  0  0  0  1  0  0 |
    Pq|  0  1  1  0  0  1 |
    S |  0  0  0  0  0  1 |
    T |  1  0  0  1  1  0 |
s
   

Resultado Final ($M_{composta}$)

    
         A  C Pç Pq S  T
       ------------------
    A |  0  1  0  1  0  1 |
    C |  1  1  0  1  0  1 |
    Pç|  0  1  1  1  0  1 |
    Pq|  0  1  1  1  0  1 |
    S |  1  1  0  1  0  1 |
    T |  1  0  0  1  1  0 |
s
   

Conclusão da Integração: A composição é robusta, permitindo que o passageiro utilize o metrô no Terminal (isolado) e, com uma baldeação para o ônibus, alcance a Avenida (M(6, 1) = 1).
